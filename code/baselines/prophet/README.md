# Prophet

Implementacao do baseline Prophet para o recorte didatico do dataset Favorita.

## Objetivo

Fornecer um baseline moderno e interpretavel para forecasting, incluindo sazonalidades e um regressor exogeno de promocao.

## Arquivos principais

- `STRUCTURE.md`: explica a arquitetura da implementacao.
- `THEORY.md`: resume o fundamento teorico do Prophet.
- `model.py`: define ajuste e previsao.
- `run.py`: executa o pipeline completo no Favorita.
- `test_model.py`: testes automatizados.
- `results/`: artefatos gerados pela execucao.

## Como executar

python /Users/anderson/Desktop/QRCNotebooks/favorita_reservoir_series/code/baselines/prophet/run.py
