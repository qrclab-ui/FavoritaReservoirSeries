# Seasonal Naives

Implementacao do baseline sazonal para o recorte didatico do dataset Favorita.

## Objetivo

Estabelecer uma referencia minima forte e transparente antes de comparar modelos mais sofisticados.

## Arquivos principais

- `STRUCTURE.md`: explica como o codigo esta organizado.
- `THEORY.md`: resume o fundamento teorico do baseline.
- `model.py`: implementa a previsao sazonal.
- `run.py`: executa o modelo no recorte Favorita.
- `test_model.py`: testes automatizados.
- `results/`: artefatos gerados pela execucao.

## Como executar

```bash
python /Users/anderson/Desktop/QRCNotebooks/favorita_reservoir_series/code/baselines/seasonal_naives/run.py
```
