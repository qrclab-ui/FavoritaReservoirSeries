# Common

Espaco para codigo compartilhado entre todos os modelos.

## Itens esperados

- leitura e validacao do dataset Favorita;
- criacao de janelas temporais;
- splits de treino, validacao e teste;
- metricas como MAE, RMSE, WAPE e MAPE;
- simulacao simples de politica de estoque para avaliacao de negocio.
