# LSTM

Implementacao de um baseline recorrente neural para o recorte didatico do Favorita.

## Objetivo

Oferecer uma comparacao sequencial treinavel, capaz de modelar memoria temporal explicitamente, em contraste com os baselines estatisticos e com o futuro pipeline de RC.

## Arquivos principais

- `STRUCTURE.md`: descreve a arquitetura do codigo.
- `THEORY.md`: resume o fundamento teorico da LSTM.
- `model.py`: define o modelo, o treino e a previsao recursiva.
- `run.py`: executa o pipeline completo.
- `test_model.py`: testes automatizados.
- `results/`: artefatos gerados pela execucao.

## Como executar

python /Users/anderson/Desktop/QRCNotebooks/favorita_reservoir_series/code/classical/lstm/run.py
