# QRC

Implementacao didatica de Quantum Reservoir Computing para previsao de demanda no Favorita usando Qiskit.

## Objetivo

Construir um benchmark quantico reproduzivel no mesmo recorte experimental dos modelos anteriores, com um reservatorio quantico de reuploading e readout linear.

## Arquivos principais

- `STRUCTURE.md`: explica a arquitetura do codigo.
- `THEORY.md`: resume o fundamento teorico da abordagem usada.
- `model.py`: define o extrator quantico de features e o readout.
- `run.py`: executa o experimento ponta a ponta.
- `test_model.py`: testes automatizados.
- `results/`: artefatos gerados pela execucao.

## Como executar

```bash
python /Users/anderson/Desktop/QRCNotebooks/favorita_reservoir_series/code/qrc/run.py
```
