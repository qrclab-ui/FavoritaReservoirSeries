# Pacote Overleaf

Este pacote foi preparado a partir de `latex_20260404_175310`.

## Como usar

1. Envie todo o conteúdo desta pasta para um projeto novo no Overleaf.
2. Defina `main.tex` como arquivo principal, se necessário.
3. Compile com `pdfLaTeX`.

## Conteúdo

- `main.tex`: arquivo principal pronto para compilação.
- `MANUSCRIPT.tex`: cópia do manuscrito principal, também normalizada para Overleaf.
- `sbc-template.sty`: template da SBC.
- `sbc.bst`: estilo bibliográfico da SBC.
- `computational_results_20260402_222902/`: figuras e tabelas locais referenciadas no artigo.

## Observação

O bloco `\input@path` usado localmente foi removido nesta versão para que a compilação funcione de forma autocontida no Overleaf.
