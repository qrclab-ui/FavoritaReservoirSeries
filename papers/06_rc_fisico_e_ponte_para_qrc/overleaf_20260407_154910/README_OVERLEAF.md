# Pacote Overleaf

Este pacote foi preparado a partir de `latex_20260407_154910`.

## Como usar

1. Envie todo o conteúdo desta pasta para um projeto novo no Overleaf.
2. Defina `main.tex` como arquivo principal, se necessário.
3. Compile com `pdfLaTeX`.

## Conteúdo

- `main.tex`: arquivo principal pronto para compilação.
- `MANUSCRIPT.tex`: cópia do manuscrito principal.
- `sbc-template.sty`: template da SBC.
- `sbc.bst`: estilo bibliográfico da SBC.
- `computational_results_20260402_222902/`: figuras e artefatos locais referenciados no artigo.

## Observação

O pacote já está autocontido e não depende de `\input@path` nem de arquivos fora desta pasta.
