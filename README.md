# Favorita Reservoir Series

Projeto-base para uma serie de 7 artigos sobre computacao de reservatorio aplicada a previsao de demanda e otimizacao de estoque com o dataset Favorita.

## Estrutura

- `dataset/`: dados brutos, externos e processados.
- `code/`: implementacoes de baselines, modelos classicos, RC e QRC.
- `papers/`: uma pasta para cada artigo da serie.

## Objetivo

Usar o mesmo problema de negocio ao longo de toda a serie:

1. prever demanda;
2. comparar familias de modelos em um protocolo temporal comum;
3. evoluir de baselines e IA classica para RC e, por fim, QRC.
