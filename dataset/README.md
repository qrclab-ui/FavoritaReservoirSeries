# Dataset Favorita

Esta pasta contem o dataset principal da serie.

## Subpastas

- `raw/`: arquivos originais baixados.
- `external/`: dados auxiliares futuros, se necessario.
- `processed/`: dados limpos, janelas temporais e splits experimentais.

## Arquivos baixados

- `train.csv`
- `test.csv`
- `stores.csv`
- `oil.csv`
- `holidays_events.csv`
- `transactions.csv`
- `sample_submission.csv`

## Origem

Os arquivos foram baixados de um espelho publico do desafio Kaggle Favorita:

- `https://huggingface.co/datasets/t4tiana/store-sales-time-series-forecasting`

O objetivo desta pasta e preservar uma copia local do dataset para os experimentos da serie.
